RABIYA ARSHAD — PORTFOLIO SITE
================================

HOW TO RUN
----------
No install, no build step. Just double-click index.html and it opens
in your browser. To put it online, the easiest free options are:

- Netlify Drop: https://app.netlify.com/drop  (drag the whole folder in)
- Vercel: https://vercel.com/new
- GitHub Pages: push this folder to a GitHub repo and enable Pages in
  Settings → Pages

FILES
-----
index.html   -> all page content (about, skills, projects, experience, contact)
styles.css   -> all styling / design
script.js    -> terminal animation, mobile menu, scroll effects

CUSTOMIZING
-----------
- Colors/fonts: all defined as CSS variables at the top of styles.css
  under ":root" — change one value and it updates everywhere.
- Text: edit directly in index.html, section by section.
- Want a photo? Add an <img> inside the .hero-copy or .about-grid div
  in index.html and style it in styles.css.
